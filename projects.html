My Projects (Ongoing)


VisNet – Lip Reading CNN Platform (C++, HTML, CSS, JavaScript)

VisNet converts video of someone speaking into text by analysing lip movements. In a team of 3, we built both the deep learning model and the interactive three-page website, creating a system that processes large video inputs locally while providing clear user guidance. This project combined AI, web development, and real-time inference in one cohesive platform.
